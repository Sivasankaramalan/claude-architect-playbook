# Content Coverage

This public guide is designed to be useful without publishing protected or employer-specific material.

## Included

- Exam blueprint overview: domains, weights, scenario themes, and study flow.
- Original domain notes for agentic architecture, MCP, Claude Code, prompt engineering, context management, and Claude APIs.
- Cheat sheets and flashcards for rapid revision.
- Architecture-pattern notes for common scenario types.
- Public-safe source summaries written as original explanations.
- Original mock questions and templates for community contributions.
- Revision plans and final-checklist material.

## Intentionally Excluded

- Real exam questions.
- Exam dumps or copied practice-bank questions.
- Private employer training notes.
- Raw PDFs, PPTX files, DOCX files, videos, and HTML exports.
- Private URLs, emails, credentials, internal system names, or proprietary campaign data.

## Known Gaps

- This guide is not a substitute for official Anthropic training or hands-on practice.
- The public question bank is intentionally small until original community questions are added.
- Some API details may change over time; verify implementation details against official public documentation before building production systems.

## Best Use

Use this repository as a structured revision companion:

1. Read `README.md` for the exam map.
2. Study each domain folder.
3. Use `10_Cheat_Sheets/One_Page_Revision.md` for daily recall.
4. Add your own original scenario questions to `09_Exam_Question_Bank/My_Mock_Questions.md`.
5. Track readiness with `14_Revision/Final_Checklist.md`.
