# Claude Code Exam Patterns

- New teammate missing config: user-level config was used instead of project-level.
- Large task executed directly: use plan mode.
- CI feedback too noisy: define explicit criteria and disable high false-positive categories.
- Stale resumed context: use fresh session with summary and targeted re-analysis.
