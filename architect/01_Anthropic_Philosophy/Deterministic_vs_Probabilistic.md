# Deterministic vs Probabilistic

## Probabilistic Controls

Prompt instructions, few-shot examples, temperature settings, and self-critique improve likelihood. They are appropriate for style, quality, preference, and ambiguous reasoning.

## Deterministic Controls

Hooks, tool call interception, schema validation, prerequisite gates, permission checks, and CI rules enforce behavior. Use them when a single failure has financial, security, compliance, or customer-trust impact.

## Exam Shortcut

If the wording says must, guarantee, prevent, enforce, compliance, security, refund, payment, transfer, or approval, the answer is usually deterministic.
