# Claude Certification Playbook

Unofficial, community-friendly study guides for Anthropic Claude certifications.

This repository currently covers two exam tracks:

| Track | Folder | Status |
| --- | --- | --- |
| Claude Certified Architect — Foundations (CCAF) | [`architect/`](./architect/) | Ready to study |
| Claude Certified Developer – Foundations (CCDV F) | [`developer/`](./developer/) | Study handbook filled — verify live API details on official docs |

## How To Navigate

1. Pick your exam track from the table above.
2. Open that folder’s `README.md` for the blueprint, study path, and golden rules.
3. Stay inside one track while preparing — domains and question banks are exam-specific.

## Public Content Policy

These guides are for public learning. Do not commit:

- Real exam questions or exam dumps
- Private employer training material
- Proprietary PDFs, slide decks, videos, or copied course content
- Credentials, private links, emails, tokens, or internal system names

Contributions should be original explanations, public-doc references, high-level summaries, and original mock questions.

See [`PUBLICATION_CHECKLIST.md`](./PUBLICATION_CHECKLIST.md) before publishing changes.

## Completeness Note

Always verify current product behavior and exam blueprints against official public Anthropic documentation. This repository is unofficial and community-maintained.
