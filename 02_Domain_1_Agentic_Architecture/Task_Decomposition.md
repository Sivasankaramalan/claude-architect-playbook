# Task Decomposition

Broad topics must be decomposed broadly before specialist work begins. A report on renewable energy that covers only solar and wind is usually a coordinator decomposition failure, not a search-agent failure.

## Patterns

- Fixed pipeline: predictable workflows such as extraction and code review.
- Dynamic decomposition: open-ended research, debugging, unfamiliar codebase exploration.
- Multi-pass: per-item local analysis plus cross-item integration.
