# Domain 1 Exam Patterns

- Missing coverage: coordinator decomposition failure.
- Unsourced synthesis: context passed without metadata.
- Compliance skip: add tool interception hook or prerequisite gate.
- Attention dilution across many files: use per-file passes plus integration pass.
