# Course Catalog Summary

Use the official public learning catalog to identify current Claude and Anthropic Academy courses relevant to the certification. Course availability and names can change, so verify against the official portal before final exam preparation.

## Recommended Learning Themes

- Claude fundamentals and responsible AI usage.
- Claude API basics, including Messages API and tool use.
- Prompt engineering for reliable outputs.
- Structured data extraction and schema design.
- Agentic systems and multi-step tool workflows.
- Claude Code configuration and development workflows.
- MCP concepts and tool integration.
- Context management, reliability, evaluation, and human review.

## How To Track Completion

For each course or module, record:

- Completion date.
- Concepts that felt unclear.
- One architecture pattern you can apply.
- One exam-style trap related to the topic.
- Follow-up practice questions to write yourself.

## Public Repo Note

Do not copy course text, screenshots, quizzes, or assessments into this repository. Summarize concepts in your own words and link to official public resources where appropriate.
